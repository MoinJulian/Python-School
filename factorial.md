What is a factorial number?

Is a number n which is represented as n! which means all positive integers for the number n from the number n to 1

Formula: n! = n x (n - 1) x ... x 1

Calculate a factorial number?

5! = 5 * 4 * 3 * 2 * 1:
    Which is equal to:
        120
